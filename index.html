Project 1 was built using strictly HTML, CSS, and JavaScript. Based off the classic game hangman, users can enter a word or phrase and have their friends guess each letter. The hangman image changes based on the number of incorrect guesses. The game displays a final win or loss image if the user wins or loses.

The art portfolio was a lab built to showcase personal artwork in one centralized location. This site has built in mobile rendering and provides a nice gallery for all paintings. Built using bootstrap to quickly produce a visually appealing prototype.

The Konjo memory game was built before WDI as a prework project using HTML, CSS, and JavaScript. After completing the required functionality, I added more features including eight cards instead of four. The logic in javascript to check flipped cards against each other flipped card was especially tricky to implement once more than four cards are used. Finally, a counter to keep score was added to track the users score after finding a match or failing to do so.

The pixel art lab was built using HTML, CSS, and Javascript by implementing DOM manipulation through mouse movements. Users are able to select a color from the menu and "paint" the area below using the mouse.
